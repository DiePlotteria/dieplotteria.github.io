# DIE Plotteria - Website

Offizielle Website für **dieplotteria.de**. 
Fokus auf beplottete Geschenke für jeden Anlass und **Plottermoments** (Plotter-Partys für JGA, Kindergeburtstage etc.).

## Struktur
- `index.html`: Hauptseite mit Galerie.
- `style.css`: Design (Schriften: Dolce Vita & Yoshi Font).
- `script.js`: Filter-Logik für Produkte & Momente.
- `/fonts`: Enthält die Marken-Schriftarten.
- `/images`: Produktfotos und Event-Bilder.

## Hosting
Diese Seite wird via **GitHub Pages** bereitgestellt.
